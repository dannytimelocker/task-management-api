# Task Management API

A simple RESTful API built using ASP.NET Core Web API.

## Features
- Create, Read, Update, Delete tasks
- Basic validation
- Clean controller structure

## Tech Stack
- ASP.NET Core Web API
- C#

## Endpoints
GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/{id}  
DELETE /api/tasks/{id}

## How to Run
1. Clone the repo
2. Open in Visual Studio
3. Run the project
4. Use Swagger to test endpoints

## Purpose
This project demonstrates backend development skills including API design, routing, and CRUD operations.
